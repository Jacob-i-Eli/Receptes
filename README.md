# 🍳 Recetario Tradicional

¡Bienvenido a este recetario digital! Aquí encontrarás una colección de recetas tradicionales explicadas paso a paso, ideales tanto para principiantes como para cocineros experimentados.

## 📋 Menú de Recetas

Selecciona una receta del menú para ver los ingredientes y el proceso de elaboración detallado:

* [🥐 Ensaimada Mallorquina Tradicional](ensaimades.md) - El dulce más icónico de Mallorca, con su característica fermentación lenta y textura hojaldrada.
* [🥘 Paella Valenciana Clásica](arrosbrut.md) - El clásico arros brut de Mallorca con pollo, conejo y más cosas.

## 📸 Contribuir

¿Tienes una receta familiar que te gustaría compartir? ¡Las contribuciones son bienvenidas!
1. Haz un fork de este repositorio.
2. Crea una nueva rama (`git checkout -b nueva-receta`).
3. Añade tu archivo Markdown en la carpeta `recetas/`.
4. Enlázalo en este `README.md`.
5. Abre un Pull Request.
